# Tic-tac-toe game
Build with react.js with vite. This game also has a history that you can use for rollback your turn into spesific moves.

### How to Run
1. Install dependencies <br>
``` npm i ```
2. Run <br>
``` npm run dev ```
3. Build <br>
``` npm run build ```
